# SolaRa

Solar System Rasterizer. Uses three.js for 3D rendering.

## Run Instructions
Go to the SolaRa folder. Use the following command:
```
npm run dev
```

## Sidenote
- There is no ring for Saturn cuz I'm too lazy to render that
- The Earth is flat cuz I said so
